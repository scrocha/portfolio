1. Sistema de Ranking de Atletas de Vela

Desenvolvi um sistema de ranking para atletas de vela em um projeto em grupo com a Confederação Brasileira de Vela. Utilizei Python com pandas e scikit-learn para analisar dados e gerar insights que melhoraram o desempenho da equipe.
